# AIS_docs

## Install:
`npm install && cd ./front-end/ && bower install`

## Launch:

1. **Server:**
    * Usual: npm start
    * With nodemon: npm run backend
1. **Development build:**
    * For Linux: npm run devLnx
    * For Windows: npm run devWin
1. **Production build:**
    * For Linux: npm run prodLnx
    * For Windows: npm run prodWin
